Module introduces the API that defines Feature Detector utilities in the dcv library.

interface Detector;
Feature detector interface.

Each feature detector algorithm implements this interface.

abstract const Feature[] detect(in Image image, size_t count);
Detector features in the image.

Params:
Image image Image in which features are looked for.
size_t count How many features are to be extracted from given image.
