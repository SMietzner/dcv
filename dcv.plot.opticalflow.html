Slice!(SliceKind.contiguous, [3], ubyte*) colorCode(Slice!(SliceKind.contiguous, [3], float*) flow, float maxSize = 0);
Draw color-coded optical flow.

Params:
Slice!(SliceKind.contiguous, [3], float*) flow Optical flow displacement vectors.
float maxSize Value which is considered to be the maximal displacement value for color saturation. Default is 0, which gets reset in the algorithm to 10% of image diagonal length.

Returns:
RGB image of color-coded optical flow.
