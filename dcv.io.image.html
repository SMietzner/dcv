struct ReadParams;
Image reading parameter package type.

Image imread(in string path, ReadParams params = ReadParams(ImageFormat.IF_UNASSIGNED, BitDepth.BD_UNASSIGNED));
Read image from the file system.

Params:
string path File system path to the image.
ReadParams params Reading parameters - desired format and depth of the image that's read. Default parameters include no convertion, but loading image orignal data depth and color format. To load original depth or format, set to UNASSIGNED (ImageFormat.IF_UNASSIGNED, BitDepth.BD_UNASSIGNED).

return:
Image read from the filesystem.

Throws:
Exception and ImageIOException from imageformats library.

bool imwrite(in string path, size_t width, size_t height, ImageFormat format, BitDepth depth, ubyte[] data);
Write image to the given path on the filesystem.

Params:
string path Path where the image will be written.
size_t width Width of the image.
size_t height Height of the image.
ImageFormat format Format of the image.
BitDepth depth Bit depth of the image.
ubyte[] data Image data in unsigned bytes.

return:
Status of the writing as bool.

bool imwrite(in Image image, in string path);
Convenience wrapper for imwrite with Image.

Params:
Image image Image to be written;
string path Path where the image will be written.

return:
Status of the writing as bool.

bool imwrite(SliceKind kind, size_t[] packs, T)(Slice!(kind, packs, T*) slice, ImageFormat format, in string path);
Convenience wrapper for imwrite with Slice type.

Params:
Slice!(kind, packs, T*) slice Slice of the image data;
ImageFormat format Explicit definition of the image format.
string path Path where the image will be written.

Returns:
Status of the writing as bool.
