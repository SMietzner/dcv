pure T linear(T, size_t N, Position...)(Slice!(N, T*) slice, Position pos) if (isNumeric!T && isScalarType!T && allSameType!Position && allSatisfy!(isNumeric, Position));
Linear interpolation.

Params:
Slice!(N, T*) slice Input slice which values are interpolated.
Position pos Position on which slice values are interpolated.

Returns:
Interpolated resulting value.
