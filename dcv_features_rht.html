dcv.features.rht

Module introduces Randomized Hough Transform implementation.

Example:
// Load an image from filesystem.
Image image = imread("/path/to/image.png");

// Calculate edges by use of canny algorithm
auto edges = image
                .sliced
                .rgb2gray
                .asType!float
                .conv(gaussian!float(1.0f, 3, 3))
                .canny!ubyte(100);

// Setup RHT line extraction context
auto lines = RhtLines().epouchs(50).iterations(250).minCurve(25);

// Lazily iterate and detect lines in pre-processed image
foreach(line; lines(edges)) {
    // do stuff with lines..
}


For more elaborated module description visit the RHT example.

template BaseRht()
A template that bootstraps a full Randomized Hough transform implementation. The basic primitives required are as follows.

Types:
Curve and Key tuples that define curve parameters and accumulator key for the curve.

Functions:
curveKey - accumulator key for a curve, onCurve - test if a point is on curve, fitCurve - fit a curve to a given random access range of points.

ref auto threshold(int threshold);
Set threshold for a curve to be considered in an accumulator

auto opCall(T)(Slice!(2, T*) image);
Run RHT using non-zero points in image as edge points.

auto opCall(T, Range)(Slice!(2, T*) image, Range points) if (isInputRange!Range);
Run RHT using prepopullated array of edge points (that may be filtered beforehand).
