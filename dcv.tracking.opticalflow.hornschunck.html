struct HornSchunckProperties;
Horn-Schunck algorithm properties.

size_t iterationCount;
How many iterations is algorithm evaluated to estimate the flow values.

float alpha;
Smoothing weight parameter for the flow field.

size_t gaussKernelSize;
Size of the gaussian kernel used to blur the image as pre-process step before the algorithm.

float gaussSigma;
Sigma value of gaussian kernel used to blur the image.

float tol;
iteration stopping criterion.

class HornSchunckFlow: dcv.tracking.opticalflow.base.DenseOpticalFlow;
Horn-Schunck algorithm implementation.

this(HornSchunckProperties props = HornSchunckProperties());
Initialize the algorithm with given set of properties.

DenseFlow evaluate(inout Image f1, inout Image f2, DenseFlow prealloc = emptySlice!([3], float), bool usePrevious = false);
Evaluate Horn-Schunck dense optical flow method between two consecutive frames.

Params:
Image f1 First image, i.e. previous frame in the video.
Image f2 Second image of same size and type as , i.e. current frame in the video.
DenseFlow prealloc Optional pre-allocated flow buffer. If provided, has to be of same size as input images are, and with 2 channels (u, v).
bool usePrevious Should the previous flow be used. If true is treated as previous flow, and has to satisfy size requirements.

Returns:
Calculated flow field.
