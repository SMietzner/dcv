dcv.features.corner.fast

Module implements FAST[1] corner detector algorithm - Features from accelerated segment test (FAST) algorithm discovered and developed by Edward Rosten and Tom Drummond.

This package offers D class interface to machine generated C code, adopted to D, which is located originally on Edward Rosten's github.

class FASTDetector: dcv.features.detector.Detector;
FAST corner detector utility.

static immutable int PERFORM_NON_MAX_SUPRESSION;
Should the non-maximum suppression be performed with the detection.

static immutable int SORT_OUT_FEATURES_BY_SCORE;
Should the features be sorted by score at the output.

enum Type: int;
Pixel neighborhood type, described in the paper.

pure nothrow @safe this(uint threshold = 100, Type type = Type.FAST_9, int flags = 0);
Default constructor.

Params:
uint threshold Threshold on pixel intensity difference between central and border pixels.
Type type Pixel neighborhood type.
int flags Flag value where PERFORM_NON_MAX_SUPRESSION and/or SORT_OUT_FEATURES_BY_SCORE flags can be passed.

const @property auto threshold();
Threshold for corner detection.

@property auto threshold(uint value);
Threshold value setter.

const @property auto type();
@property auto type(Type type);
Type of the detector.

const @property auto flags();
Algorithm flags.

const Feature[] detect(in Image image, size_t count = 0);
Detect features for given image.

Params:
Image image Input image where corners are to be found. Only 8-bit mono image is supported as this time.
size_t count How many corners are to be found.

Returns:
Array of found feature points.
